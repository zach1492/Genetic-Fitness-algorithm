# Genetic Algorithm Word Solver

A simple Python project that uses a genetic algorithm to evolve random strings into the target phrase. Struggles on large phrases

## Features
- Randomly generated agents
- Fitness evaluation system
- Parent crossover
- Dynamic mutation rate
- Evolution across generations

## How It Works
The program generates random strings and scores them based on how many characters match the target phrase. The best-performing strings are selected to create the next generation until the correct phrase is found.

## Technologies
- Python
- Random module
- String module
